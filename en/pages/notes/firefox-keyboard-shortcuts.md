# Firefox most useful keyboard shortcuts

source: "./firefox-keyboard-shortcuts-source.html"

## Navigation

| Command                | Shortcut         |
|------------------------|------------------|
| Back                   | Ctrl + [         |
| Forward                | Ctrl + ]         |
| Reload                 | Ctrl + r         |
| Reload (override cache | Ctrl + Shift + r |
| Stop                   | Esc              |

## Windows and Tabs

| Command             | Shortcut           |
|---------------------|--------------------|
| Close tab           | Ctrl + w           |
| Close window        | Ctrl + shift + w   |
| New tab             | Ctrl + t           |
| New Window          | Ctrl + n           |
| Next tab            | Ctrl + tab         |
| prev tab            | Ctrl + shift + tab |
| Select Location bar | Ctrl + l           |


## Current Page

| Command | Shortcut |
|---------|----------|
| Zoom in | Ctrl +   |
| Zoom in | Ctrl -   |
